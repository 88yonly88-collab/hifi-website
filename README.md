# 株式会社Hi-Fi コーポレートサイト

株式会社Hi-Fi（HIGH FIDELITY INC.）のコーポレートサイトです。
業務効率化ツール開発・ワークフロー簡易化を主軸に、SNS集客・動画編集・非属人制YouTube運営までAIを活用してワンストップで支援します。

## 構成

| ファイル | 内容 |
|---|---|
| `index.html` | トップページ（単一HTML・自己完結。外部依存はGoogle Fontsのみ） |
| `docs/site-design-document.md` | サイト設計書（コンセプト・IA・デザインシステム・実装方針） |

## デザイン方針

- **トーン**：ウォーム・フレンドリー（顧客に寄り添う温かい印象）／信頼 × 先進
- **配色**：クリーム基調＋コーラル＋ピーチ＋ゴールド
- **書体**：Zen Maru Gothic / Zen Kaku Gothic New / Fraunces

## ローカルでの確認

```bash
# このディレクトリで簡易サーバーを起動
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

または `index.html` をブラウザで直接開く。
