# 株式会社Hi-Fi コーポレートサイト

株式会社Hi-Fi（HIGH FIDELITY INC.）のコーポレートサイトです。
業務効率化ツール開発・ワークフロー設計を主軸に、SNS集客・動画制作・YouTube運営までAIを活用してワンストップで支援します。

## 構成

| ファイル | 内容 |
|---|---|
| `index.html` | トップページ（単一HTML・自己完結。外部依存はGoogle Fontsのみ） |
| `concepts/` | リニューアル検討時のコンセプト案（A: ダーク精密系 / B: エディトリアル系 / C: デモ主導型 / D: 採用ハイブリッド案） |
| `docs/site-design-document.md` | 旧サイト設計書（初版時点のもの） |

## デザイン方針（2026-07 リニューアル）

- **コンセプト**：「見せる先端」。サイバー表現を避け、AIに馴染みのない銀行・法人クライアントにも構えさせない上質さと、実際に動く仕組みのデモで技術力を伝える
- **トーン**：エディトリアル×実演。明朝体の見出し＋生成りの紙色＋深緑のアクセント
- **配色**：生成り `#F7F5EF` ／ 墨 `#211E18` ／ 深緑 `#0E6B5C`
- **書体**：Shippori Mincho B1（見出し）／ Zen Kaku Gothic New（本文）／ Inter（欧文ラベル）
- **アイコン**：絵文字は不使用。線画SVGのみ
- **ヒーローのデモパネル**：自動化ワークフローの実例を、専門用語を使わない日常語のステップで表示。ステップが自動で進行するループアニメーション付き

## ローカルでの確認

```bash
# このディレクトリで簡易サーバーを起動
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

または `index.html` をブラウザで直接開く。
